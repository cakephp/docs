Introdução
###############

O framework CakePHP fornece uma base sólida para sua aplicação.
Ele pode lidar com todos os aspectos, da solicitação inicial do usuário todos os
caminho para a renderização final de uma página web. E uma vez que o framework
segue os princípios do MVC, ele permite que você personalize facilmente
e estenda a maioria dos aspectos de sua aplicação.

O framework também fornece uma estrutura organizacional básica, de
nomes de arquivos para nomes de tabelas de banco de dados, mantendo toda a sua aplicação
consistente e lógica. Este conceito é simples mas poderoso. Seguindo
as convenções você sempre saberá exatamente onde as coisas estão e
como elas estão organizadas.
