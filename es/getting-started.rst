PRIMEROS PASOS CON CAKEPHP 2.0
##########################

El framework CakePHP te proporciona una base robusta que aporta estructura para tu aplicación.
Te permite gestionar todos los aspectos, desde que el usuario realiza el request inicial hasta la manera en la que se genera la página web de respuesta.
CakePHP sigue el principio MVC (Modelo Vista Controlador), lo que te permite adaptarlo y extenderlo con facilidad para que funcione según tus necesidades.

El framework te facilita una estructura organizativa básica, para los nombres de los ficheros, los nombres de las tablas en la base de datos, etc. Lo que aporta consistencia a tu aplicación. Estos conceptos son simples y poderosos. Sique las convenciones y siempre sabrás dónde está cada elemento y cómo se organizan de forma intuitiva.

Lo mejor para aprender CakePHP es ponerse manos a la obra y desarrollar algo concreto. Para empezar vamos a construir un blog de forma muy sencilla.
