3.x Guide de Migration
######################

.. toctree::
   :hidden:

Les guides de migration contiennent des informations sur les nouvelles
fonctionnalités introduites dans chaque version et le chemin de migration entre
les versions 2.x et 3.x. Si vous utilisez actuellement la version 1.x, vous
devrez d'abord mettre à jour vers la version 2.x. Consultez la documentation
de la version 2.x pour avoir les guides de mise à jour qui s'y rapportent.
